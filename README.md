# E-Commerce Sales Analytics using MySQL

## Project Overview

This project analyzes e-commerce sales data using MySQL to generate business insights related to revenue performance, customer behavior, product performance, and regional sales trends.

## Tools Used

* MySQL
* MySQL Workbench
* SQL
* Draw.io

## Database Schema

The project consists of three tables:

* Customers
* Products
* Orders

The Orders table links Customers and Products through foreign keys.

## Business Questions Solved

1. Total Revenue Analysis
2. Product Revenue Analysis
3. Customer Revenue Analysis
4. Category Revenue Analysis
5. State Revenue Analysis
6. Top Products Analysis
7. Average Order Value
8. Repeat Customer Analysis
9. Monthly Revenue Trend
10. Highest Revenue Customer
11. Unsold Products Analysis
12. Customer Lifetime Value
13. Revenue Contribution Analysis
14. Top Product by Category
15. Month-over-Month Growth Analysis
16. Customer Segmentation
17. Pareto Analysis
18. RFM Analysis
19. State-wise Market Share Analysis

## Key Skills Demonstrated

* Joins
* Aggregate Functions
* Subqueries

* Window Functions
* Customer Analytics
* Revenue Analytics

## Project Outcome

The analysis identified top-performing customers, products, categories, and regions while providing actionable recommendations to improve customer retention, revenue growth, and operational decision-making.
